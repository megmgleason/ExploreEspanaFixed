How do I choose a User?
    The user can be chosen by doing a random trivia game, rock paper scissors, or a sword fight.


How should the User start?
    If I'm not there to set it up, have the User complete the tutorial so they know how to work the controllers. However, have them SKIP the last step so that they can't save a location or see a previously saved location. Or you could just wing it. It's important that I (or the teacher) choose the correct place to start.


How does the User move?
    Following the tutorial, the user must use only the RIGHT contoller. Toggle the back button to move. The user does NOT need to leave the ground for the scavenger hunt, or else they will get very confused.


Can the rest of the class be involved?
    Cálmate, the rest of the class will participate. There will be prompts and opprotunites to interact, and students should try to make the most of their part.


What are the cards for?
    The cards are for the other students to have a part in the experience. When prompted, the student should do the action/read the prompt on the card.


What should I do if the user gets lost?
    Great question. If they get off track, guide them back on the course. If all else fails have the class close their eyes and have me fix it. A note for the user: please don't fly. It'll make it harder for all of us.


What else should I do?
    Explore España should be as interactive as possible, so if the teacher has any ideas on how to involve more of the class, we're open to any opprotunites to improve.
